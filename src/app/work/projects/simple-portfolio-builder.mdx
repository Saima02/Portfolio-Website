## Overview

The Student Performance Dashboard is an **interactive Excel-based tool** designed to analyze student performance across subjects. It provides a **visual summary of grades, top-performing students, result distribution, and subject averages** using pivot tables, slicers, and charts.

## Key Features

- **Top 10 Students Analysis:**  
  Horizontal bar chart displaying the top 10 students, automatically updating with slicers.

- **Result Distribution (Pass vs Fail):**  
  Pie chart showing percentage split linked to dataset filters.

- **Grade Distribution:**  
  Column chart showing the count of each grade to understand performance trends.

- **Subject-wise Average Marks:**  
  Pie chart showing averages of Math, Physics, and Chemistry.

- **Interactive Slicers:**  
  Grade and Subject slicers for **one-click filtering** of the entire dashboard.

## How to Use

1. Download the `.xlsx` file.  
2. Open in Microsoft Excel (2016 or later recommended).  
3. Use slicers to filter by grade or subject.  
4. Analyze results interactively with charts and pivot tables.

## Technologies Used

- **Platform:** Microsoft Excel  
- **Features:** Pivot tables, slicers, charts, conditional formatting  

## Challenges and Learnings

- Organizing large datasets for effective pivot table analysis.  
- Creating interactive charts and slicers for dynamic reporting.  
- Ensuring clarity and usability for easy insights.
